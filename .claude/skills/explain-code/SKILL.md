Al exlicar código, incluye:

1.- **Empieza con una analogía** Compara el código con algo de la vida diaria.
2.- **Dibuja un diagrama** Usa ASCCI para mostrar flujo, estructura o relaciones.
3.- **Walk-through paso a paso** Explica qué pasa en cada línea importante
4.- **Un gotcha común** Un error o malentendido frecuenta.
5.- **Alternativas de la implementación** Describe por qué se utlizó esta implementación y qué otras opciones había.

Mantén las explicaciones conversacionales. Para conceptos complejos, usa múltiples analogías.
